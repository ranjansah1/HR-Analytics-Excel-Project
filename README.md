# HR Analytics Dashboard — Excel Project

![Dashboard Preview](dashboard_preview.png)

## Project Overview
A comprehensive HR Analytics project analyzing 1,470 IBM employees 
across Salary, Performance, Hiring Trends, and Attrition dimensions 
using Microsoft Excel.

## Key Insights
- Sales dept has highest avg salary (Rs.6,959/month)
- Overtime employees show 15% higher attrition rate
- 41% workforce is in 26-35 age group (young workforce)
- Gender pay gap is less than 2% across all job levels

## Tools Used
- Microsoft Excel (Pivot Tables, Power Query, IF/COUNTIF formulas)
- Interactive Dashboard with KPI Cards and 5 Charts
- IBM HR Analytics Dataset (Kaggle) — 1,470 rows, 35 columns

## Dashboard Features
- 4 KPI Cards (Employees, Salary, Attrition Rate, Performance)
- Avg Salary by Department (Bar Chart)
- Performance Distribution (Donut Chart)
- Age Distribution (Column Chart)
- Gender Pay by Job Level (Line Chart)
- Overtime vs Attrition (Bar Chart)
